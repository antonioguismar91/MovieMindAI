# MovieMindAI
Recomendador personal de películas con IA. Incluye gestión de historial en CSV, sistema de filtrado de películas ya vistas ("lista negra"), votación por estrellas y una biblioteca personalizada de títulos vistos. Desarrollado con Python, Streamlit y Groq API.
